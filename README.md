# 2023-numble-challenge-web-novel

> Springboot로 웹소설 서비스 클론코딩하기 Spring boot 클론코딩
> 
> ex ) 네이버 시리즈

## 요구 사항

- 소설 등록/작성/삭제
- 선호작 목록 
  (소설의 마지막 화, 내가 읽은 마지막 화를 함께 조회할 수 있어야 합니다. ex. 274/291 )
- 다양한 기준의 소설 목록 조회 기능(홈 노출/베스트/기타)
- 소설 편당 조회 및 결제
- 구매수 집계 등
- 포인트 충전
